# AI Health Assistant - SDG 3

## Overview
AI Health Assistant is a web-based healthcare support platform developed to support United Nations Sustainable Development Goal 3 (Good Health and Well-Being). The system helps users manage their health through an AI chatbot, symptom checker, medicine reminders, health tips, and emergency information.

## SDG Selected
SDG 3 - Good Health and Well-Being

## Problem Statement
Many people forget medications, lack access to reliable health information, and often ignore early symptoms. This can lead to preventable health complications and reduced awareness of preventive healthcare practices.

## Features
- AI Health Chatbot
- Symptom Checker
- Medicine Reminder System
- Daily Health Tips
- Emergency Information Section
- Responsive Dashboard

## Technology Stack
- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a browser.
